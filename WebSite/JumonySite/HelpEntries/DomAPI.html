DOM API即是修改文档结构的API，通过DOM API可以方便的在文档中新增元素、注释或是文本节点，也能够将一小段HTML片段插入文档，或是在在文档中移除这些元素。Jumony 提供了标准统一的DOM API用于修改文档结构，对于所有支持修改的HTML DOM，这些API提供了一致的行为。

针对 IHtmlContainer 的 API

IHtmlElement AddElement( [int index], string elementName )
返回值：

返回添加在DOM中的元素

参数:

elementName: 元素名

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

IHtmlTextNode AddTextNode( [int index], string htmlText )
返回值：

返回添加在DOM中的文本节点

参数:

htmlText: HTML文本

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

IHtmlComment AddComment( [int index], string comment )
返回值：

返回添加在DOM中的注释节点

参数:

comment: 注释内容

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾

IEnumerable<IHtmlNode> AddFragment( [int index], IHtmlFragment fragment )
返回值：

返回添加在DOM中的节点列表

参数:

fragment: 要添加的文档碎片

index: 要添加的位置，此参数可以省略，若省略index参数，则添加到容器的末尾
